Buttons

Descripción

Buttons: Botones personalizados Bootstrap para acciones en formularios, diálogos, y más con soporte para multiples tamaños, estados, y más.

Bootstrap incluye varios estilos predifinidos de botones, cada uno serviendo a un propósito semántico, con muchas extras para más control.



Las clases .btn son diseñadas para ser usadas con el elemento button. Sin embargo, éstas clases pueden ser usadas con el elemento o el el elemento input. Cuando se usan las clases de botón en un elemento "a" son usadas para accionar alguna funcionalidad de la página (como collapsar contenido), más enlazar hacia nuevas páginas o secciones dentro de la página, éstos enlace deben tener un role="button" para apropiadamente cumplir sus propósitos para la tecnología asistiva como los lectores de pantalla.
