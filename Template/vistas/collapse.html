Collapse

Descripción

Alterna la visibilidad de contenido en el proyecto con pocas clases y plugins JavaScript de boostrap.
El plugin JavaScript de collapse es usado para mostrar y ocultar contenido. Botones o anchors son usados para accionarlos hacia específicos elementos que se han mapeado.
