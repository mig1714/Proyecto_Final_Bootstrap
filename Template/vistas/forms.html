Forms

Descripción

Uso de clases Bootstrap para optimizar los formularios y mostrarlos de una forma mejorada en los buscadores y dispositivos. Debe usarse un atributo con un tipo apropiado para los inputs (e.g., email para dirección de correo o number para información numérica) para poder tomar ventaja de los nuevos controles de input como la verificación de correo, selección de número y más.
