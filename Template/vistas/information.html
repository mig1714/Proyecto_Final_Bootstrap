Breadcrumb

Descripción

Breadcrumb: Indica la ubicación de la página actual dentro de una jerarquía de navegación y automáticamente agrega un separador vía CSS.

Use unal lista ordenada o desordenada con enlaces a los elementos para crear un breadcrumb con un mínimo de estilo. Use las utilidades para agregar estilos adicionales.

Divisores son agregados automáticamente en CSS por medio de ::before y contenido. Ellos puede modificar una propiedad local CSS mediante --bs-breadcrumb-divider, o en $breadcrumb-divider Sass variable — y $breadcrumb-divider-flipped para RTL. Por defecto ponemos nuestra variable Sass, la cual se configura como un fallback de la propiedad. De ésta forma, obtienes un divisor global que puede anular sin recompilar CSS en cualquier momento.

Pagination

Descripción

Pagination: indica una serie de contenido relacionado que existe através de múltiples páginas.

Se usa una gran cantidad enlaces para nuestra paginación, haciendo que los enlaces difícilmente se pierdan y es fácil de escalar mientras provee áreas grandes para tocar. La paginación es construído listas HTML así que los lectores de pantalla pueden anunciar el número de enlaces disponibles. Envolviendo con un elemento

para identificar como una sección de navegación para los screen readers y otra tecnología asistiva.
